03 — Projects

Selected work

Group Project Jan – Apr 2025
CIFAR-100 Image Classification
with SE-Attention ResNet
Problem
General-purpose classifiers treat all feature channels equally, limiting focus on semantically meaningful patterns across 100 fine-grained object categories.
Data
CIFAR-100 benchmark — 60,000 colour images (32×32 px), 100 classes, 50K training / 10K test split.
Approach
ResNet-18 backbone augmented with Squeeze-and-Excitation (SE) attention blocks in PyTorch 2.0. Three ablation experiments: loss functions, learning rate schedules, batch size effects.
My Contribution
Led SE block integration, designed and ran all ablation experiments, produced evaluation visualisations, co-authored the final report.
Outcome / Impact
~72% top-1 accuracy and AUC 0.99 on the test set. Label smoothing + cosine LR schedule consistently improved validation performance across all runs.
