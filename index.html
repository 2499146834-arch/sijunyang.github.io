03 — Projects

Selected work

Group Project Mar – Apr 2026
Real-Time Smart Parking Detection — Dual-Model YOLO Framework
Problem
Drivers searching for parking account for ~30% of urban traffic congestion. Single-model CV approaches fail to generalise across different parking lot viewpoints and lighting conditions.
Data
Custom dataset of 2,850 annotated images (1920×1080) across day/night, rain/fog conditions — 15,420 bounding box labels across 4 classes (Legal, Available, Illegal, Full).
Approach
Dual-model adaptive framework: YOLOv8n with 396 predefined ROIs for known viewpoints + YOLO11n for arbitrary perspectives. Intelligent view detection module auto-selects model per scene. Deployed with web interface and RESTful API.
My Contribution
System testing, performance evaluation, and benchmarking. Co-authored the final paper covering methodology and experimental results.
Outcome / Impact
94% detection accuracy · 92% precision · 93% recall — 22% improvement over traditional CV. Real-time 42 FPS at 24ms latency on Full HD video. Maintained >70% accuracy under challenging weather conditions.
Group Project Mar – Apr 2026
CIFAR-100 Image Classification — ResNet-18 with SE-Attention
Problem
Standard CNNs treat all feature channels equally, limiting discrimination across 100 fine-grained categories (animals, vehicles, objects) sharing low-level visual features.
Data
CIFAR-100 benchmark — 60,000 colour images (32×32 px), 100 classes, 50K training / 10K test split. Trained on CUDA GPU with PyTorch 2.0.
Approach
ImageNet-pretrained ResNet-18 with a custom SE attention block inserted after the feature extractor. Three ablation experiments: loss functions (CrossEntropy vs Label Smoothing), LR schedules (step vs cosine), and batch size effects on convergence.
My Contribution
Results visualisation and report writing. Generated all evaluation plots (ROC curve, loss/LR/batch size comparison charts) and co-authored the final project report.
Outcome / Impact
~72% top-1 accuracy · AUC 0.99 on test set. Label smoothing + cosine LR schedule consistently outperformed baseline across all experimental runs.

My Contribution

Outcome / Impact

Group Project Feb – Mar 2026
MovieRec — Collaborative Filtering with Z-Score Optimisation
Problem
MovieLens dataset has 99.51% matrix sparsity — only 0.49% of user-item pairs have ratings, making similarity computation unreliable for most user pairs.
Data
MovieLens dataset — 6,040 users, 3,706 movies. Most ratings cluster around 6–7 on a 0–10 scale, with strong positive bias requiring normalisation.
Approach
Implemented User-Based CF (UBCF) and Item-Based CF (IBCF) with Z-Score standardisation to correct rating bias. Deployed a full-stack Flask web application with pandas backend and interactive UI.
My Contribution
Led evaluation and visualisation (Part 3 of 5). Designed performance comparison charts, error distribution analysis, and algorithm strength benchmarking across UBCF vs IBCF.
Outcome / Impact
Item-Based CF achieved 59% lower MAE than User-Based CF (0.4410 vs 1.0786). Z-Score standardisation improved prediction accuracy across both methods. IBCF also delivered broader genre diversity (11.90 vs 6.20 avg genres/user).
Group Project Feb – Mar 2026
Dogs vs Cats Image Classification — CNN with Interactive Demo
Problem
Binary image classification across highly variable real-world photos — different breeds, lighting, poses, backgrounds, and partial views — requiring robust generalisation.
Data
Kaggle Dogs vs Cats dataset — 25,000 balanced JPEG images (12,500 per class). Split 80/20: 20,000 training / 5,000 validation. Resized to 150×150 px.
Approach
CNN built with TensorFlow/Keras. Data augmentation (rotation, flip, zoom, shear) to reduce overfitting. Optimisation experiments across Dropout, BatchNorm, and optimiser selection. Gradio interactive demo for real-time inference.
My Contribution
Led performance evaluation and error analysis (Member D). Produced confusion matrix, per-class accuracy breakdown, and error case review. Identified key failure modes in ambiguous or low-quality images.
Outcome / Impact
89% validation accuracy on 5,000 held-out images with symmetric performance across both classes. Interactive Gradio demo deployed for real-time image upload and classification.
Group Project Oct – Nov 2025
Cooling System Data Analysis — Association Rules & Classification
Problem
Data centre cooling systems operate across high-load and energy-saving modes. The goal was to extract actionable operational rules and predict cooling strategies from sensor data.
Data
Cooling system sensor logs including ambient temperature, server load, chiller usage rate, inlet temperature, and power consumption readings.
Approach
Applied Apriori association rule mining to discover operational patterns, and a Decision Tree classifier for strategy prediction. Evaluated rules by confidence, lift, and support; benchmarked classifier with F1, Kappa, and accuracy.
My Contribution
Association rule analysis, business insight interpretation, and report writing. Identified high-priority operational rules across temperature control, high-load, and energy-saving scenarios.
Outcome / Impact
Extracted 10 high-value association rules (confidence ≥93%, lift ≥2.7). Defined optimal operating temperature range (22.73–25.24°C) and load thresholds for energy-saving mode. Decision tree identified as unusable (accuracy 21–25%), steering team toward rule-based recommendations.
